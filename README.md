# WeChatShare

Typecho 微信分享插件

使用此插件可方便地自定义微信和QQ分享链接中的标题、描述、小图标和URL，让你的分享链接不再单调

## 注意：由于微信更新，如果你想要在微信中直接分享，请按以下步骤操作：

> 1、公众号通过微信认证
> 2、填写 AppID 和 AppSecret （开发 > 基本配置）
> 3、添加服务器IP 到 IP白名单中 （开发 > 基本配置 > IP白名单）
> 4、添加域名到 JS安全域名中（设置 > 公众号设置 > 功能设置 > JS接口安全域名）
